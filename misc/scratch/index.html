How This Works

When text is pasted into the text area of the Scratch Pad, scripts are triggered to ensure the text is appropriate for the editor. The following checks and corrections are available or automatically performed:

  1. Foreign characters, such as hidden or differently encoded characters, are removed.
  2. Non-standard punctuation is converted to standard forms. For instance, the double quote ‚Äù is changed to ".
  3. List bullet characters at the beginning of lines are removed:
    1. Unordered list markers are removed automatically.
    2. Ordered list markers can be removed manually by clicking the "Remove OL Markers" button.
  4. Multiple spaces between words within a paragraph are converted to a single space.
  5. Empty lines between paragraphs are removed.
  6. The "Remove Leading Spaces" button removes any extra spaces at the beginning of paragraphs, which often occur when copying a list. This feature ensures that no unintended spaces are included in the content.
  7. The "Remove OL Markers" button removes alphanumeric list markers that can be inadvertently copied when transferring an ordered list. This includes list markers such as 1), 1., a., and a).
    By using this feature, you can quickly clean up your text, ensuring that no extraneous list markers are pasted into the content.

  9. "Paragraph Spacing" toggle ‐ When activated, this toggle adds an extra space between paragraphs pasted into the text area, ensuring clear and consistent paragraph separation.

    • Automatic Deactivation: The toggle automatically turns off after a single use, preventing unintended formatting changes.

  11. "Fix Fragment" Button ‐ This feature addresses cases where answer choices are pasted into the text area as full sentences (with a capital letter at the beginning and/or punctuation at the end) when they should actually be fragments. The text will be processed to lowercase the first letter of each line and remove any punctuation at the end.

The validation process continues to evolve as new characters are discovered and added. If you encounter any issues that the validation does not address, please inform me so I can update the validation process accordingly.

The validation also checks for any HTML tags that might be mistakenly copied. While it alerts the user to the presence of HTML tags, it does not remove them automatically, as there are instances where HTML tags may be necessary.
