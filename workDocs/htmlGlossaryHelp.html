Adding to Vocab/Glossary for HTML Courses

  1. Download the list given in the bug ticket
  2. Verify the audio has been created and filenames match what is in the sheet
  3. Place the audio in the appropriate media folders if not already placed
  4. Download the glossary sheet from the course docs in box, if it already exists
    • Make a new file if it doesn't already exist
      "MBID Glossary.xlsx"
  5. Add the new terms from the provided sheet into the sheet from box
    • Make sure the terms are in the correct order and formatted appropriately
      • Master Build Order
      Unit # Assignment #      		 Word Filename
      Path to vocab audio in media drive      		 Definition
      3011 0101 Vocab Term 3011\3011-01\3011-01-01\3011-01-01-vocab\3011-01-01-vocabterm.mp3 a list or collection of words or of words and phrases usually alphabetically arranged and explained or defined
      • The Order column should be sequential, and the Order column format in the spreadsheet should be set to text so it keeps the leading zeros.
        0101
        0101
        0102
        0103
        0103
        etc.
      • When updating, insert a new row where appropriate to keep the sequential order. Using the example above, if you need to add a new vocab term for 0102 then you would insert a new row between 0101 and 0102 or between 0102 and 0103 and then add in the data for the new vocab term for 0102.
  6. Once all the terms have been added and audio has been placed:
    1. Save the data in the spread sheet from/for Box
    2. Open the CMT, click on "Glossary Import"
    3. Add the spread sheet from Box, click upload
  7. After the spread sheet has been uploaded go to Publish in the CMT and publish the course to Sandbox.
