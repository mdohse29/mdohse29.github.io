Remove Leading Space

Update!

There is some new functionality to the ScratchPad. The following functions are active:

  • List bullet removal - if there are bullets that copy over from a list they will/should be removed automatically. There will not be an alert for this.
  • NEW Extra space removal - If there are empty lines in the text that is pasted they will be removed automatically. This is to keep from adding extra empty paragraphs when copying into CAT.
  • NEW Remove leading space button - I added a button that will remove the empty space in front of lines, or paragraphs.

If anyone sees any formatting or HTML elements that are not detected please let me know. Send me whatever examples you can and I will review and update the functionality if possible.
