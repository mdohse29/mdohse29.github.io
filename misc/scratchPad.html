Update!

There is some new functionality to the ScratchPad. The following functions are active:

  • HTML tag detection - if there are hidden HTML styles/formatting in the text there will be an alert. Currently HTML is only detected and not removed, since there is a possibility there could be a reason to have HTML in the text I want to see how often it occurs.
  • List bullet removal - if there are bullets that copy over from a list they will/should be removed automatically. There will not be an alert for this.
  • NEW Remove extra space button - I added a button that will remove extra space between paragraphs to avoid adding empty paragraphs in CAT.
    If you think this would be better as automatic and not a button let me know and I will change it.

If anyone sees any formatting or HTML elements that are not detected please let me know. Send me whatever examples you can and I will review and update the functionality if possible.
